**Feekool** is a fully Open Source, privacy-first forum software designed for anonymous discussions. Post without accounts, track users, or revealing personal data for sensitive topics, communities, or quick brainstorming sessions.

## Key Features
- **100% Anonymous**: No passwords, emails, or IP logging. Threads and replies stay truly private.
- **Infinitely Customizable**: Adapt to *any* themes, tech support, local events, or niche hobbies. Swap themes, logos, and categories in minutes via simple config files.
- **AI-Powered Setup**: Launch your forum in **2 minutes** flat. Use our AI generator: input your topic, and it spits out a ready-to-deploy instance with custom branding, rules, and structure.
- **Self-Hosted & Free**: MIT-licensed, runs on any server (Docker, Vercel, or VPS). No vendor lock-in, zero costs.
- **Modern & Fast**: Responsive design, real-time updates, search, moderation tools, and infinite threading. Built with React + Markdown for blazing speed.
